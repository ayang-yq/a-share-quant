# A股量化交易

## 择时体系 v4.3.1
- 价格动量 BB (70%) + 融资动量 BB (30%)
- 风格轮动 v2: 申万小/大盘 60日分位
- 三态判定 + 迟滞带
- 月度检查 + 3% 漂移阈值
- 回测 (2012-2026, 14年): 年化 11.8%, 夏普 0.73, 最大回撤 -27.4%

## 目录结构
- `backtest/` — 回测脚本 (择时/风格/止损/估值等)
- `scripts/`  — Cron 定时脚本 (仓位监控/股票池review等)
- `screening/` — 选股筛选脚本
- `docs/`     — 投资框架文档

## 技术栈
- Python 3.13 + AKShare + Pandas
- 数据源: 东财API / 申万指数 / 蛋卷
