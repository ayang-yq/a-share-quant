# A股景气度投资框架 v1.0

> 景气度投资的本质：不猜底、不抄顶，只跟踪"谁在变好"，
> 在景气拐点右侧上车，在景气见顶回落时下车。
> 与价值投资相反——价值投资问"便宜吗"，景气度投资问"在变好吗"。

---

## 一、核心哲学

```
价值投资：便宜 → 买 → 等均值回归（左侧）
景气度投资：变好 → 买 → 跟着趋势走（右侧）
```

- 不追求买在最低点，追求买在趋势确认的右侧
- 允许贵，但不允许增速放缓
- 持有的理由是"还在变好"，卖出的理由是"不再变好了"

**适用场景**：
- A股的高波动+高换手环境，景气度投资的胜率比纯价值更高
- 非全职投资者无法深度调研基本面，宏观/行业景气指标是更好的跟踪工具
- 与择时体系互补：择时管总仓位，景气度管行业/个股选择

## 二、景气度三层跟踪体系

### 第一层：宏观景气（决定总仓位方向）

| 指标 | 数据源(AKShare) | 频率 | 景气判断 |
|------|----------------|------|---------|
| 制造业PMI | `macro_china_pmi()` | 月度 | >50扩张，连续3月上行=景气回升 |
| M1-M2剪刀差 | `macro_china_money_supply()` | 月度 | M1增速>M2=资金活化，利好股市 |
| 社融存量增速 | 东财手动/央行 | 月度 | 社融拐点领先盈利拐点6-9个月 |
| PPI同比 | 东财手动 | 月度 | PPI上行=上游周期景气 |
| CPI同比 | `macro_china_cpi_monthly()` | 月度 | CPI上行=消费景气，过高=通胀风险 |
| LPR | `macro_china_lpr()` | 月度 | 降息=宽货币，利好成长股估值 |
| 人民币汇率 | 东财 | 日度 | 贬值利空外资流入，升值利好 |

**宏观景气评分规则**：
- 每个指标计分：+1（扩张/利好）/ 0（中性）/ -1（收缩/利空）
- 7个指标总分范围 -7 ~ +7
- 总分 >= +4：宏观景气扩张，进攻仓位可偏高
- 总分 0~+3：中性
- 总分 <= -1：宏观景气收缩，降低仓位

### 第二层：行业景气（决定行业配置）

#### A. 行业景气度打分卡（月度更新）

对申万31个一级行业，每个行业跟踪以下维度：

| 维度 | 数据源 | 说明 |
|------|--------|------|
| 行业指数60日涨跌幅 | `index_analysis_daily_sw()` | 价格是最终的结果，涨=市场认可景气 |
| 行业PE分位(5年) | 本地积累申万PE时序 | 低PE分位+涨=戴维斯双击，高PE分位+涨=纯情绪 |
| 营收增速(YoY) | 东财datacenter个股汇总 | 行业龙头营收增速的平均/中位数 |
| 利润增速(YoY) | 东财datacenter个股汇总 | 扣非净利润增速，比净利润更真实 |
| 机构资金流向 | `stock_individual_fund_flow()` | 行业内龙头股主力资金净流入 |
| 北向资金行业配置 | 东财 | 外资偏好行业的边际变化 |

#### B. 行业景气度评分（满分100）

```
价格动量（60日涨跌幅排名）:  25分
盈利增速（利润YoY排名）:     25分
估值吸引力（PE分位倒数排名）: 20分
资金流向（主力净流入排名）:   15分
相对强度（vs沪深300超额）:   15分
```

每月更新，选Top 5行业作为进攻方向。

#### C. 行业景气轮动规律（A股经验）

```
复苏初期：  金融地产 → 建筑建材 → 汽车
复苏中期：  科技（电子/计算机）→ 制造（机械/电力设备）
复苏后期：  消费（食品饮料/医药）→ 周期（有色/煤炭）
滞胀期：    能源/农业/黄金
衰退期：    公用事业/红利/必需消费
```

### 第三层：个股景气（决定具体买什么）

**景个股气度核心指标**：

| 指标 | 计算方式 | 景气判断 |
|------|---------|---------|
| 营收增速趋势 | 连续4个季度营收YoY | 加速>20% 或企稳回升=景气 |
| 扣非净利润增速 | 连续4个季度扣非YoY | 同上 |
| 毛利率趋势 | 4个季度毛利率变化 | 毛利率提升=定价权增强 |
| 机构持仓变化 | 基金季报持仓变动 | 机构增仓=认可景气 |
| 经营现金流/净利润 | 比值>0.8 | 利润含金量高 |

**个股筛选流程**：
1. 从行业景气Top 5中选个股
2. 营收+扣非净利润连续2个季度加速或企稳
3. 毛利率不恶化
4. 现金流健康（经营CF/净利润 > 0.7）
5. 排除：负债率>60%、应收账款增速远超营收增速

## 三、景气度投资的买卖规则

### 买入规则

```
右侧买入，三重确认：

1. 行业景气评分进入Top 5（月度确认）
2. 个股季报营收+利润增速企稳/加速（季报确认）
3. 价格站上20日均线且20日均线拐头向上（技术确认）

三重确认不全满足 = 观察，不买
```

### 卖出规则（比买入更重要）

```
景气度投资最容易犯的错：赚了不走，坐过山车

卖出信号（任一触发）：
1. 行业景气评分跌出Top 10（连续2个月）
2. 个股营收增速连续2个季度下滑
3. 毛利率连续2个季度恶化
4. 机构大幅减仓（基金持仓下降>5pp）
5. 止损线：从买入价回撤15%

不要等"景气完全结束"才走——景气投资的核心是吃鱼身
```

### 仓位管理

```
单行业最大仓位：20%
单只个股最大仓位：5%（观察池）/ 8%（核心持仓）
景气度越强的行业，仓位越重
景气度衰减的行业，逐月减仓，不一次性清仓
```

## 四、景气度投资 vs 价值投资 对照

| 维度 | 价值投资 | 景气度投资 |
|------|---------|-----------|
| 核心问题 | 便宜吗 | 在变好吗 |
| 买入时机 | 左侧（估值低） | 右侧（趋势确认） |
| 持有理由 | 等均值回归 | 跟着景气走 |
| 卖出理由 | 贵了 | 增速放缓 |
| 估值态度 | PE越低越好 | PE可以高，但增速要匹配 |
| 风险偏好 | 低风险 | 中等风险 |
| 持有周期 | 长（1-3年） | 中（3-12个月） |
| 适用标的 | 稳定盈利的价值股 | 增长明确的成长股/周期股 |
| 最大陷阱 | 价值陷阱（便宜但有原因） | 增长陷阱（增速放缓但PE还高） |

**关键insight**：两者不是互斥的，可以融合——
用景气度选方向（哪个行业在变好），用估值管买卖（便宜时买、贵了减）。
你的择时体系其实就是这个思路的体现。

## 五、A股景气度投资的特殊注意事项

### 1. 政策景气 > 基本面景气

A股最大的景气驱动力往往是政策，不是基本面：
- 新能源：2020-2021双碳政策驱动，行业增速40%+
- AI/TMT：2023 ChatGPT催化，算力产业链暴增
- 地产：2009/2015/2022三次政策刺激均带来板块行情

**规则**：政策利好宣布后1-3个月内，行业景气度会显著上行。
跟踪政策渠道：中央经济工作会议、国务院常务会议、行业五年规划。

### 2. 景气度有惯性但会突变

景气度趋势通常持续3-6个季度，但政策转向（集采、双减、地产三道红线）
可以瞬间改变景气方向。不能只看过去外推。

**规则**：每季度财报季必须重新评估景气度，不能"躺赢"。

### 3. 一致预期是反向指标

记忆中的教训：券商一致性预期在行业下行期系统性高估。
当所有研报都说"行业景气度高"时，往往是景气度的顶部区域。

**规则**：景气度判断以一手数据（财报、PMI、价格数据）为准，
研报/一致预期只做参考，不作为决策依据。

### 4. 景气度投资要回避的雷

- **增收不增利**：营收涨但利润率在压缩（价格战信号）
- **景气扩散到垃圾股**：行业内最差的公司都在涨=过热
- **估值透支2年增速**：PE/增速 > 2，即使景气也在透支
- **单一事件驱动**：一张订单/一个合同带动整个行业的"景气"往往是假的

## 六、与现有体系的集成方案

你的v4.3.1体系是仓位管理+择时框架，景气度投资可以嵌入为**行业/个股选择层**：

```
v4.3.1 现有架构：
  总仓位 → 风格(大/小) → ETF配置（红利/300/2000/进攻）

加入景气度后的架构：
  总仓位 → 风格(大/小) → 行业景气Top5 → 个股/ETF选择
                              ↑ 新增这一层
```

具体集成点：
1. **"进攻"类别的内部配置**：不用固定比例，改为按行业景气度动态分配
2. **个股池更新**：每季度用景气度打分卡重新评估45只股票池
3. **新增/移除个股**：景气度进入Top5的行业中选股替换景气度下滑的标的

## 七、数据获取工程方案

### 自动化数据源（AKShare）

```python
# 宏观景气（月度更新）
pmi = ak.macro_china_pmi()                          # PMI
m2 = ak.macro_china_money_supply()                   # M2/M1剪刀差
cpi = ak.macro_china_cpi_monthly()                   # CPI

# 行业景气（每日/月度更新）
sw_info = ak.sw_index_first_info()                    # 31行业截面PE/PB
sw_hist = ak.index_analysis_daily_sw(                 # 行业PE时序
    symbol='一级行业', start_date=..., end_date=...)

# 个股景气（季报后更新）
financial = ak.stock_financial_abstract_ths(          # 业绩摘要
    symbol='300015', indicator='业绩报表')
```

### 手动数据源（需browser/东财）

| 数据 | 来源 | 频率 |
|------|------|------|
| 社融存量增速 | 央行/东财 | 月度 |
| PPI同比 | 国家统计局 | 月度 |
| 北向资金行业配置 | 东财 | 日度 |
| 基金持仓变动 | 天天基金/东财 | 季度 |
| 行业政策动向 | 国务院/发改委 | 不定期 |

### 建议做成Cron Job

```
每月第一个交易日：更新宏观景气评分 + 行业景气排名
每季度财报季结束后：更新个股景气度打分
结果推送飞书
```
